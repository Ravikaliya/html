JavaScript Array Functions

push()

Adds one or more elements to the end of an array.

const array = [1, 2, 3];
array.push(4);
console.log(array); // Output: [1, 2, 3, 4]

pop()

Removes the last element from an array.

const array = [1, 2, 3];
array.pop();
console.log(array); // Output: [1, 2, 3]

shift()

Removes the first element from an array.

const array = [1, 2, 3];
array.shift();
console.log(array); // Output: [2, 3]

unshift()

Adds one or more elements to the beginning of an array.

const array = [1, 2, 3];
array.unshift(0);
console.log(array); // Output: [0, 1, 2, 3]

concat()

Merges two or more arrays.

const array1 = [1, 2];
const array2 = [3, 4];
const newArray = array1.concat(array2);
console.log(newArray); // Output: [1, 2, 3, 4]

slice()

Returns a shallow copy of a portion of an array.

const array = [1, 2, 3, 4];
const newArray = array.slice(1, 3);
console.log(newArray); // Output: [2, 3]

splice()

Changes the content of an array by removing, replacing, or adding elements.

const array = [1, 2, 3, 4];
array.splice(1, 2, 'a', 'b');
console.log(array); // Output: [1, 'a', 'b', 4]

indexOf()

Returns the first index at which a given element can be found.

const array = [1, 2, 3, 2];
const index = array.indexOf(2);
console.log(index); // Output: 1

lastIndexOf()

Returns the last index at which a given element can be found.

const array = [1, 2, 3, 2];
const index = array.lastIndexOf(2);
console.log(index); // Output: 3

forEach()

Executes a function once for each array element.

const array = [1, 2, 3];
array.forEach(element => console.log(element));
// Output: 1
//         2
//         3

map()

Creates a new array populated with the results of calling a provided function on every element.

const array = [1, 2, 3];
const newArray = array.map(element => element * 2);
console.log(newArray); // Output: [2, 4, 6]

filter()

Creates a new array with all elements that pass the test implemented by the provided function

reduce()

Executes a reducer function on each element of the array, resulting in a single output value.

some()

Tests whether at least one element in the array passes the test.

every()

Tests whether all elements in the array pass the test.

find()

Returns the first element that passes a test.

findIndex()

Returns the index of the first element that passes a test.

includes()

Determines whether an array includes a certain value.

join()

Joins all elements of an array into a string.

reverse()

Reverses the order of the elements in an array.

sort()

Sorts the elements of an array.

fill()

Fills all the elements in an array with a static value.

flat()

Creates a new array with all sub-array elements concatenated into it.

flatMap()

Maps each element using a mapping function, then flattens the result into a new array.
